# Steps to create your own project showcase
1. Make a folder named my projects
2. Open your folder in V.S code
3. Make the files that is present in my folder (of this repo)
4. Copy the code and paste according to the files
5. Enter your projects' name and url given in the space in the code

   Now, your project showcase is ready now add this in your portfolio to showcase all your skills and project at one place
